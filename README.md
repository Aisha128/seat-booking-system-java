# 🎟️ Seat Booking System (Java)

A simple console-based seat booking system built using Java.

## 🚀 Features
- Reserve seats
- Cancel bookings
- View available seats

## 🛠️ Technologies
- Java
- File Handling

## ▶️ How to Run
1. Open in Eclipse or VS Code
2. Run `MainMenu.java`
